=================
Speak Text
=================

*Speaks the given text with the system's text-to-speech voice*

* **Input:** Can be used for the *Text* parameter
* **Output:** Same as input

Parameters
----------

* **Text:** The text that should be spoken
* **Wait until Finished:** Determines whether the next action starts immediately, or when speaking has finished.
