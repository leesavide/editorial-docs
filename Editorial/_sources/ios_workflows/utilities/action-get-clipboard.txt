=================
Get Clipboard
=================

*Passes the text in the system clipboard to the next action*

Note that instead of using this action, it is often more convenient to use the built-in `Clipboard` variable directly in parameters of other actions.

* **Input:** N/A
* **Output:** The contents of the clipboard

Parameters
----------

This action has no configurable parameters.
