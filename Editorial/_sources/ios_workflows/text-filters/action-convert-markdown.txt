========================
Convert Markdown to HTML
========================

*Converts Markdown text (given as input) to HTML*

The *Preview* panel uses the same kind of conversion, but the output of this action doesn't include any (CSS) styling.

There are various parameters to control whether extended Markdown syntax (like footnotes) should be interpreted.

* **Input:** The Markdown source text
* **Output:** The converted HTML
