=================
URL Escape
=================

*Converts special characters in the input, so that it can be used as a parameter in a URL*

This replacement is also called "percent encoding" (e.g. ``%20`` for spaces).

* **Input:** The text to filter
* **Output:** The input text with special characters replaced by their percent-encoded equivalents

Parameters
----------

This action has no configurable parameters.
