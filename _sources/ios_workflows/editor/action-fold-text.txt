=========
Fold Text
=========

*Folds (hides) parts of the current document*

* **Input:** Can be used as a variable in the `Range` parameter
* **Output:** Same as input

Parameters
----------

* **Fold:** What to fold -- can be `Selection` or `Range of Text`
* **Range:** If the `range` mode is selected, this specifies a character range in the document that should be folded
* **Placeholder Label:** A short piece of text that is shown as a placeholder for folded text -- tapping on the label unfolds the text.
