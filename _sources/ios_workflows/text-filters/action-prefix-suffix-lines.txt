=====================
Prefix / Suffix Lines
=====================

*Adds text before or after each line of the input*

* **Input:** Can be used in the `Text` parameter
* **Output:** The text with the prefix/suffix added to each line

Parameters
----------

* **Prefix/Suffix:** `Prefix` adds the text `before` each line, `Suffix` after each line
* **Text:** The text that is added to each line.
